英会話教師：マイク・ニマ（Mike Nima）

About Mike

  • コーネル大学英文科卒業
  • オックスフォード大学英文科修士号
  • カリフォルニア大学バークレー校 元英文学講師
  • 室蘭工業大学、 札幌医科大学 元英語講師

教師であるマイクはアメリカのペンシルベニア州生まれワシントン D.C.育ち。16歳からニューヨーク州、イギリス、カリフォルニア州の様々な大学で英米文学を専門として勉強。アメリカ留学中の日本人の家庭教師としての経験も豊富。

2009年に来日、2013年に英語教室を開校。趣味は、ジョギング、水泳、ヨガ、瞑想、カラオケです。

One-to-One Lessons

お知らせ＜2021年3月更新＞

マイク英会話教室札幌

Mike's English Class Sapporo

札幌でネイティブ教師の英会話個人レッスンを受けたい方は、今すぐマイク英会話教室へ！

大人の方に向けて一人ひとりのニーズにお応えしたマンツーマンレッスンを行っております。英会話はもちろん英文作成・翻訳・資料作成にも対応。 実際に使える英語スキルが盛りだくさん！

当スクールでは豊富なコースをご用意！目的に合わせて学べます

  • 日常英会話
  • 仕事での英語・接客英語
  • 海外留学向けの英会話レッスン
  • ビジネス英語
  • アカデミック英語
  • TOEIC・TOEFL・IELTS対策
  • 英検・英語試験対策
  • 英語でのプレゼンテーション・ワークショップ・カンファレンス・会議対策
  • トラベル英語・旅行英会話
  • 英語ライティング
  • 英訳・英語翻訳・英語通訳
  • 英語の歌詞・英語作詞

本当に使える英語力を身に付けたい方はぜひ Mike's English Class Sapporo にお越しくださいませ。
